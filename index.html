What does it take to land a job as a Data Scientist?

Project Summary


Using Beautiful Soup, we scraped Indeed.com to extract pertinent data related to the given job title: “Data Scientist.” We then loaded this JSON data into a structured database, Postgres SQL, where we cleaned the data.

We conducted Natural Language Processing (NLP) using Scikit-Learn and PySpark to extract raw textual data, tokenize that data, filter for stopwords, and run TF-IDF analyses to understand the importance of specific words within job descriptions for the given job: “Data Scientist”.

Using Tableau, we made unique visualizations, dashboards, and stories. Highlights of our discoveries have posted onto a GitHub Pages website with various infographics to build upon our analyses.
